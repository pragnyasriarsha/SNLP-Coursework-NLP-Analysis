# SNLP Coursework 1 - NLP Biography Analysis

## Overview
This project performs NLP analysis on biography text using Python, SpaCy, and NLTK.

## Features
- Tokenization
- POS Tagging
- Named Entity Recognition
- Dependency Parsing
- N-gram Extraction

## Technologies
- Python
- SpaCy
- NLTK
- Scikit-learn
